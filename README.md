# Sales Performance Dashboard

Interactive dashboard to analyze retail sales:
monthly trends, sales-rep performance,
category contribution, and sales by region.

## Tech Stack
- Python (pandas, plotly, streamlit)
- Excel & Google Sheets (data + pivot)

## How to Run
pip install -r requirements.txt
streamlit run dashboard.py

## Key Insights
- Electronics accounts for ~45% of revenue.
- Sales dipped in March → a seasonal promo is needed.
